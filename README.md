## About The Project

A simple React Dapp for drop-listing NFT's


### Built With

* [React.js](https://reactjs.org/)
* [Solidity](https://soliditylang.org/)

<!-- GETTING STARTED -->

## Getting Started

1. Clone the repo
   ```sh
   git clone https://github.com/ehsnai/nft-droplist-dapp
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Start development with
   ```sh
   npm start
   ```
